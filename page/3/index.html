欧拉函数
欧拉函数φ(a)是定义在正整数上的函数，它在正整数a上的值等于序列0，1，2…，a-1中与a互质的数的个数。

若a是质数，则φ(a)=a-1

若m1，m2是两个互质的正整数，则φ(m1m2)=φ(m1)·φ(m2)

a=p^α，p是质数，则φ(a)=( p^α ) - [ p^(α-1) ] = ( p-1 )[ p^(α-1) ]

设a=( p1^α1 )( p2^α2 )…( pk^αk )，则φ(a)=a( 1-1/p1 )( 1-1/p2 ) … ( 1-1/pk )

PWN笔记（一）——保护机制

发表于 2020-03-20 | 分类于 PWN入门

操作系统提供了许多安全机制来尝试降低或阻止缓冲区溢出攻击带来的安全风险，包括DEP、ASLR等。在编写漏洞利用代码的时候，需要特别注意目标进程是否开启了DEP（Linux下对应NX）、ASLR（Linux下对应PIE）等机制，例如存在DEP（NX）的话就不能直接执行栈上的数据，存在ASLR的话各个系统调用的地址就是随机化的。
