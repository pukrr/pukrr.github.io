代理与通道

发表于 2019-11-05 | 分类于 内网渗透

代理与通道技术在内网渗透和线下AWD攻防中也中有巨大作用，本文用于介绍其中工具和应用场景。

代理

正向代理

借助一台外网机器转发流量到内网。外网机器又两块网卡，一块内网网卡，一块公网网卡。

反向代理

假设内网被种了木马，但从外网无法访问到内网木马，或者内网机器只开放某一个特定的端口，而这个特定端口又不能直接访问。所以需要内网主动连接外网，使内网流量转发到外网。
